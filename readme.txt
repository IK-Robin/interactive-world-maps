Add an interactive world map to your WordPress site! Customize colors, links, and hover descriptions easily through an intuitive admin panel. Display your map anywhere using a simple shortcode.

== Description ==

**Interactive World Maps** lets you create and customize fully responsive, interactive SVG maps for your WordPress website. No coding knowledge required!  

### Features
- Interactive world map with 6 clickable continents.  
- Responsive and mobile-friendly design.  
- Fully customizable (colors, links, hover descriptions & images).  
- Easy-to-use admin panel.  
- Works seamlessly on iOS and Android devices.  
- Embed the map in any page or post using the shortcode `[ikr_world_map]`.  

 

### Additional Features
- Custom hover tooltips.  
- Clickable regions with customizable actions.  
- Easy integration with shortcode.  


### Privacy Information  
This plugin uses SVG maps with Bootstrap and vanilla JavaScript, hosted locally without external dependencies, ensuring no sensitive data is collected and providing a secure, self-contained mapping solution. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/interactive-world-maps` directory, or install the plugin through the WordPress plugins screen directly.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Go to ** Interactive World Maps > dashboard ** to customize your map.  
4. Use the shortcode `[ikr_world_map]` to embed the map in any page or post.
